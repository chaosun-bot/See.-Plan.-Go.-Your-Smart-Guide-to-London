London's Sites by Borough
London is a city brimming with history, culture, and natural charm. Each borough has its own tourism profile — such as Big Ben in Westminster or cultural museums in Kensington and Chelsea — reflecting the city's rich and varied character.

Conclusion

This project supports the recovery of the UK tourism industry after the pandemic by creating an intelligent and interactive travel planning system for London. To meet the rising demand for urban mobility and cultural activities, the platform combines several key functions to improve both performance and usability:

• AI-based itinerary generation, map navigation, and nearby place search work together to provide routes based on time and location preferences.

• A simple and unified interface brings together different types of information, solving issues like scattered sources and limited personalisation.

• The system uses web tools, geospatial visualisation, and interface design to build a platform that is both technically sound and easy to use.

• Key tourism data—such as visitor numbers, spending, and average stay—is shown through interactive charts to help explain cultural and spatial trends in the city.

These features together support smarter travel planning by linking behaviour and location data, showing how people, places, and movement are connected. Based on the theme of the "City Networks", tourist flows are seen as changing links between people, places, and digital tools. Although the current version includes all core functions, it still relies on static data and fixed route logic. Future updates could include real-time data, flexible route planning, and feedback features to better reflect how travel patterns change and to offer more tailored, context-aware plans.
